## 📝 게시판 프로젝트 구현

Vite + React + Spring Boot 기반의 CRUD 게시판 프로젝트입니다.  
게시판 글 작성, 수정, 삭제, 조회 기능을 제공합니다.

---

## 📌 기술 스택

### 💻 Frontend
- **Vite**
- **React**
- **JavaScript**
- **Axios**

### 🔧 Backend
- **Java 17+**
- **Spring Boot**
- **JPA (Hibernate)**
- **Lombok**

### 🗄️ Database
- **MySQL 8+**

---


### 주요 기능
- 게시글 목록 조회
- 게시글 작성
- 게시글 수정
- 게시글 삭제
- 게시글 상세 조회
